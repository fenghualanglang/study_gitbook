插件搜索

输入gitbook

  • gitbook-plugin-donateGitbook 捐赠打赏按钮插件

根目录下创建book.json

{
    "title": "编写gitbook电子书教程",
    "description": "gitbook电子书建教程",
    "author": "zhangjunbo",
    "language": "zh-hans",
    "root": ".",

    "plugins": [  # 插件名字
        "donate",
        "github-buttons@2.1.0",
        "edit-link"
    ],


    "pluginsConfig": {
        "donate": {
          "wechat": "https://cloud.githubusercontent.com/assets/1890238/15489630/fccbb9cc-2193-11e6-9fed-b93c59d6ef37.png",
          "alipay": "https://pics1.baidu.com/feed/8c1001e93901213f4149ebb3b58f25d62e2e95d3.jpeg?token=cf0f8b153f1cab257749396668be2e40",
          "title": "默认空",
          "button": "打赏",
          "alipayText": "支付宝打赏",
          "wechatText": "微信捐赠"
        },
        "github-buttons": {
            "repo": "fenghuall/study",
            "types": ["start"],
            "size": "small"
        },
        "edit-link": {
            "base": "https://github.com/twbs/bootstrap/master",
            "label": "Edit This Page"
        }
    }
}

以上配置完成后，安装插件

gitbook install
